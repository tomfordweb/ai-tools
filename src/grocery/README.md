# Dependencies

```
sudo pacman -S tesseract tesserat-data-eng
```

# Help

### Could not find the Qt platform plugin 'wayland'

Try:

```
export QT_QPA_PLATFORM=wayland

```

or

```
export QT_QPA_PLATFORM=xcb
```
